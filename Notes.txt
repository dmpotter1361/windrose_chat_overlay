Step 4: How to Test It in VS Code
Before you compile anything or move it to your server, you can test the entire pipeline right there on your desktop.

Start the Server: In your VS Code terminal, type node server.js and hit Enter. You should see it boot up and connect to Discord.

Start the Client: Open a second terminal tab in VS Code (click the + icon in the terminal window) and type python client.py. You should see it connect to the local server.

The Simulation: * Open Notepad on your computer.

Save the file as Windrose.txt (this temporarily tricks the Python script into thinking the Notepad window is the game, because "Windrose" is now in the window title).

Click inside the Notepad window so it is active, and press Enter.

The custom UI box should instantly pop up! Type a message, hit OK, and watch it flow through your VS Code terminal straight into your Discord server.

Once you confirm everything is talking to each other smoothly, you'll be ready to change the IP address to your server and compile them into .exe files!
